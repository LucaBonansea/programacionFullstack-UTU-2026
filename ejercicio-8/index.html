Notas para explicar en clase

  • Bootstrap se usa con container, row y col para adaptar las secciones a celulares, tablets y laptops.
  • MediaPipe Pose detecta puntos del cuerpo con la camara y calcula angulos para contar repeticiones.
  • La Web Speech API escucha comandos como empezar, pausar, reiniciar y terminar.
  • LocalStorage guarda configuracion, repeticiones, records e historial aunque se cierre el navegador.
  • El responsive design se completa con media queries comentadas en el archivo CSS.
