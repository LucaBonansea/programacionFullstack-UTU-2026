# Guia breve para presentar

## Presentacion corta del proyecto

Hola, nuestro proyecto se llama FitVoice Arena. Es una aplicacion web responsive de entrenamiento que cuenta repeticiones con la camara, guarda el progreso del usuario y tiene una interfaz inspirada en videojuegos arcade.

## Que problema o idea resuelve

La idea fue hacer una app de ejercicio mas interactiva y creativa que una pagina comun, usando vision por computadora, diseno llamativo y una experiencia adaptable a celular y escritorio.

## Que tecnologias usamos

- HTML
- CSS
- JavaScript
- Bootstrap
- MediaPipe Pose
- LocalStorage

## Como funciona

1. El usuario elige una rutina.
2. Presiona iniciar.
3. La app activa la camara.
4. Detecta puntos del cuerpo.
5. Calcula el movimiento y cuenta repeticiones.
6. Guarda la sesion en el historial.

## Como hicimos el responsive

- Usamos Bootstrap con `container`, `row` y `col`.
- Tambien usamos media queries en CSS para ajustar tamanos, distribucion de paneles, botones, camara y temporizador.

## Que parte fue creativa

- La estetica visual tipo arena de combate.
- La mezcla entre fitness, camara y estilo arcade.
- La experiencia de usar la app como si fuera una interfaz de videojuego.

## Si preguntan por la IA

Usamos IA como apoyo en mejoras de diseno, orden del codigo, correccion de errores y documentacion, pero la idea del proyecto, las decisiones principales y la explicacion del funcionamiento fueron del grupo.

## Si preguntan por dificultades

- permisos de camara y microfono en iOS
- soporte desigual de comandos de voz segun el navegador
- mantener el conteo correcto sin romper el responsive

## Cierre

Elegimos este proyecto porque nos permitia combinar creatividad, diseno, logica y una funcionalidad distinta a una pagina tradicional.
